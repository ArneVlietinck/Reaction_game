Variable Documentation

◆ action

bool action

Boolean which representate the action of a player. When action is true, the player did an action. When action is false, the player didn't do an action.

◆ difficulty

int difficulty

Integer which representate the difficulty level of the game. Higher integer means higher degree of difficulty. It affects the time between the blinking leds.

◆ game

bool game

Boolean which representate the state of the game. If game==1, the current game is finished. Elseif game==0, the current game is still going.

◆ interaction

bool interaction

Boolean which representate the need of a players' interaction. When interaction is true, the player must do something (eg: press the user button) to avoid a game over.
